# Тестовое задание

### Устновка

Backend

```
cd ./backend/
npm install
```

Создать .env файл

```
DB_USERNAME=
DB_PASSWORD=
DATABASE=
```

Создать базу данных и пользователя и заполнить .env
Frontend

```
cd ./frontend/
npm install
```

### Запуск

Frontend

```
cd ./frontend/
npm start
```

Открыть http://localhost:3000/

Backend

```
cd ./backend/
npm run dev
```

Открыть http://localhost:3001/api/items/
